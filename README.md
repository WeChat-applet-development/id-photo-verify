证件照研究院
==================

id-photo-verify 应用功能分两大块：
1.证件照检测功能：上传手机本地照片去检测是否符合证件照使用标准。
2.证件照制作功能：现场用手机拍照去制作证件照，支持手机下载制作好的证件照或到pc网页端提取。
